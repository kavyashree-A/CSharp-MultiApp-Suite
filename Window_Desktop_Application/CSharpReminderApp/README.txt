Reminder App - Windows Forms (.NET 4.8)
--------------------------------------
How to open:
1. Unzip the folder.
2. Open the .csproj in Visual Studio 2019/2022.
3. Build and Run.

Features:
- Set reminder message and time.
- App checks every second and shows alert popup.
- Simple, modern, and user-friendly.

Notes:
- Make sure .NET Framework 4.8 Developer Pack is installed.
- The app uses Timer to check system time.
