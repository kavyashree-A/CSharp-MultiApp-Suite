# NotesKeeperApp (Sticky Notes Style)

Open this folder in Visual Studio (2019/2022) and run. Project targets .NET Framework 4.8.

Files:
- Program.cs
- Note.cs
- Form1.cs
- Form1.Designer.cs
- notes.txt (created/updated at runtime)

Notes:
- The app stores notes in notes.txt using '|||' as a separator.
- Left panel is the editor (sticky-note colored), right panel lists titles and shows small previews.
- You can Add, Edit, Delete, and Search notes.
