# ContactBookApp (Modern UI)

Open this folder in Visual Studio (2019/2022) and run. Project targets .NET Framework 4.8 style WinForms app.

Files:
- Program.cs
- Contact.cs
- Form1.cs
- Form1.Designer.cs
- contacts.txt (empty - will be created/updated at runtime)

Notes:
- Add/Edit behavior: double-click a contact to load into fields. 'Add' will append a new entry.
- The app saves to contacts.txt in the application folder when you click Save.
