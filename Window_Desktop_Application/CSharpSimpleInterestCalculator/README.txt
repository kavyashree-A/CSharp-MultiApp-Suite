Simple Interest Calculator - Windows Forms (.NET 4.8)
---------------------------------------------------
How to open:
1. Unzip the folder.
2. Open the .csproj in Visual Studio 2019/2022.
3. Build and Run.

Notes:
- Project targets .NET Framework 4.8 (net48). Make sure .NET Framework 4.8 developer pack is installed.
- Controls: enter Principal, Rate (annual percent), Time (years). Click Calculate.
